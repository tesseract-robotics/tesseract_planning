◆ start_instruction_

value_type tesseract_planning::CompositeInstruction::start_instruction_ { NullInstruction() }
private

The start instruction to use for composite instruction. This should be of type PlanInstruction or MoveInstruction but is stored as type Instruction because it is not required.

If not provided, the planner should use the current state of the robot is used and defined as fixed.
