\doxysubsection{Detailed Description}
Tesseract OMPL Weighted State Sampler. 

This allows you to provided a weight factor different joint when sampling. This is useful when you have a gantry with two linear axis with a robot attached. When sampling near you may want to scale down the rail sampling.
