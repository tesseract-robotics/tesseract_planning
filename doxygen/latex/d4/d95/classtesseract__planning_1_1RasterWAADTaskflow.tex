\doxysubsection{Detailed Description}
This class provides a process manager for a raster process. 

Given a \mbox{\hyperlink{structtesseract__planning_1_1TaskInput}{Task\+Input}} in the correct format, it handles the creation of the process dependencies and uses Taskflow to execute them efficiently in a parallel based on those dependencies.

The required format is below. Note that a transition is planned from both the start and end of each raster to allow for skipping of rasters without replanning. This logic must be handled in the execute process.

Composite \{ Composite -\/ from start Composite -\/ Raster segment (e.\+g. approach, raster, departure) \{ Composite approach Composite process Composite departure \} Composite -\/ Transitions Composite -\/ Raster segment \{ Composite approach Composite process Composite departure \} Composite -\/ Transitions Composite -\/ Raster segment \{ Composite approach Composite process Composite departure \} Composite -\/ to end \}
