\doxysubsection{Detailed Description}
Taskflow Container. 

Taskflow composition does not allow connecting to tasks within another taskflow. This is to support similar functionality where all tasks are added to the highest level taskflow and stores all tasks from a generator and the identified input and output tasks for composition
